02 / Notes

Observed structure

This is a reading instrument more than a strategy engine: it compresses market structure into a compact panel of regime, momentum and imbalance.

  • The original script is a Pine v6 overlay indicator with a table panel and alertconditions, centered on Base MA, CH1/CH2 relationships, retests and FVGs.
  • The original documentation is already unusually complete, including architecture, usage logic, asset templates and examples.
  • The `v2-codex` branch is not a rewrite; it appears to be a structured extension of the same indicator.
  • The documented v5.1 additions include macro trend status, session filtering, stop management via CH2, FVG/MA confluence highlighting and volatility warning.
  • The promotional image is not literal UI, but it reinforces the indicator's semantic core: regime, channels, FVGs, momentum and retests.

04 / Differences

v5.0 versus v5.1 branch

  • v5.0 defines a 21-row table panel and focuses on regime, channel state, retests, separation strength and FVG logic.
  • v5.1 expands the panel to 24 rows and adds macro trend via higher timeframe data using request.security.
  • v5.1 adds a stop line based on CH2, a session filter, FVG highlight logic for Base MA confluence, and a volatility warning threshold.
  • The overall architecture remains indicator-first rather than strategy-first; even the improvements are about context quality and decision filtering.
