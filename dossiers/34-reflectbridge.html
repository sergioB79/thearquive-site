01 / Technical Reading

What exists today

Project type
A narrow integration bridge, not a user-facing app.
Framework
Flask with CORS enabled for browser-origin flexibility.
Routing
Two routes: /reflect for note creation and /ping for health checks.
Security model
The bridge validates a shared secret from either a header or query parameter before forwarding to Reflect.
Integration role
Its job is simple: receive title/body payloads and convert them into Reflect note API calls.
Deployment
The comments and launch text show explicit intent to run behind Tailscale Funnel.
