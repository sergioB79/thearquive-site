01 / Technical Reading

What exists today

Project type
A camera-based interaction experiment.
Framework
A single Python script using OpenCV, NumPy, and direct webcam access.
Routing
No route or screen framework exists; the whole experience runs inside one procedural game loop.
Game model
The script captures a background frame, performs thresholded motion detection, and treats overlap with a target circle as a successful hit.
Persistence
Results are printed to the terminal only; no saved profile, score file, or UI shell exists.
Deployment read
This is a local machine experiment with no packaging or distribution setup.
