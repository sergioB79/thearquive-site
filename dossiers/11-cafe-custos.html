01 / Technical Reading

What exists today

Project type
A self-contained editorial microsite with a second-page interactive calculator rather than a single one-off article.
Framework
No framework. Both surfaces are plain HTML/CSS/JS with Google Fonts and inline styling.
Routing
The landing page index.html leads into cafe-custos.html, which holds the cost projection tool.
Styling
The entry page uses a warm magazine aesthetic with heavy typography and grain; the calculator shifts to a darker terminal-like palette.
Interaction model
The calculator uses sliders for daily quantity, out-of-home price, capsule price, and inflation, then renders year-by-year projections for mixed, outside-only, or home-only scenarios.
Sharing layer
Open Graph tags, share buttons, and a counter API call are wired into the landing page, which suggests a real intention to circulate the page publicly.

02 / Notes

Observed structure

The project is strongest where rhetoric and utility meet: first persuade the reader that the habit matters, then let the reader test their own numbers immediately.

  • The opening page is not written like a fintech tool; it is written like a newspaper feature.
  • The calculator does not exist in isolation — it is framed as the quantitative proof layer of the essay above it.
  • GitHub Pages URLs are already embedded in the metadata, so this copy was prepared with public distribution in mind.
  • The folder is small, finished enough to be usable, but still hand-built and monolithic.
