01 / Technical Reading

What exists today

Project type
A recurring trading-analysis publishing system with static public output.
Framework
The public repo is static HTML and JSON, while the README documents a Python plus Modal pipeline used to generate the dated analysis issues.
Routing
The archive entry point is index.html, which links to dated issue files such as 2026-05-14.html; latest.json stores the current consolidated output.
Styling system
Presentation is authored directly inside the generated HTML pages with dark report styling and screenshot-linked asset sections.
Pipeline note
The documented generation flow includes screenshot upload, asset and timeframe classification, per-chart analysis, cross-asset consolidation, and GitHub Pages publishing.
Archive note
This repo preserves the published surface and the operating doctrine, but not the full local pipeline code described in the README.
